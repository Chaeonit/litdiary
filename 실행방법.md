# LitDiary 실행 방법

## 프로젝트 위치
```
E:\creai+it\diary\
```

## 사전 준비

`.env` 파일이 `diary` 폴더 안에 있어야 합니다.
```
OPENROUTER_API_KEY=sk-or-...
```
OpenRouter 키는 https://openrouter.ai/keys 에서 발급받으세요.

---

## 실행 순서

**1. 터미널에서 가상환경 폴더로 이동**
```bash
cd "E:\creai+it\week2_session_0316\creaiit-api-session--main"
```

**2. 가상환경 활성화**
```bash
# PowerShell
.\.venv\Scripts\Activate.ps1

# CMD
.\.venv\Scripts\activate.bat
```

**3. diary 폴더로 이동**
```bash
cd "E:\creai+it\diary"
```

**4. 서버 실행**
```bash
python app.py
```

**5. 브라우저에서 열기**
```
http://127.0.0.1:5000
```

**6. 종료할 때**
```
Ctrl + C
```
